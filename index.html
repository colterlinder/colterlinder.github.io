Publishing a static site with GitHub Pages is easy:

  1. Go you your github acccount (ie: github.com/colterlinder -- that's mine)
  2. Make a repo called user.github.io (ie: colterlinder.github.io -- in my case)
  3. Commit a file named index.html -- that will be your homepage
  4. Now visit the site user.github.io in your broswer. It may take a minute to show up, but that's it!
